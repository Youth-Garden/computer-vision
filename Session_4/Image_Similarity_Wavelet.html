II> L thuyt: Hash Wavelet

Hash Wavelet l k thut to m bm (fingerprint) cho nh da trn tn s, gip tm kim nh tng ng nhanh chng. Quy trnh:

  1. Bin i Wavelet (DWT): Phn tch nh thnh cc bng tn (xp x v chi tit). Thng dng bng tn thp nht (LL - Low-Low) i din cho cu trc tng quan.
  2. Lng t ha: Chuyn cc h s thc thnh nh phn (0 hoc 1), thng da trn gi tr trung bnh hoc trung v.
  3. Hm bm: Ghp cc bit li thnh chui bm.
  4. So snh: Dng khong cch Hamming (m s bit khc nhau).

def load_image(path, size=(256, 256)):
    if not os.path.exists(path):

    img = cv2.imread(path, cv2.IMREAD_GRAYSCALE)

def wavelet_hash(image, wavelet='db4', level=3):
    # Chuyn i sang float  tnh ton
    im_float = image.astype(float)
    
    # Phn tch Wavelet
    coeffs = pywt.wavedec2(im_float, wavelet, level=level)
    
    # Ly thnh phn xp x (Approximation)  mc thp nht (coeffs[0])
    LL = coeffs[0]
    
    # Lng t ha: So snh vi gi tr trung v (median)
    median_val = np.median(LL)
    
    # To hash binary: 1 nu > median, 0 nu <= median
    hash_matrix = (LL > median_val).astype(int)
    
    # Lm phng thnh vector
    return hash_matrix.flatten()

def hamming_distance(hash1, hash2):
    if len(hash1) != len(hash2):
        raise ValueError("Hash lengths must be equal")
    return np.sum(hash1 != hash2)

# Kim tra hm
h_orig = wavelet_hash(dataset["Original"])
h_rot = wavelet_hash(dataset["Rotated"])

print(f"Hash length: {len(h_orig)} bits")
print(f"Hamming Distance (Original vs Rotated): {hamming_distance(h_orig, h_rot)}")

results = {}
base_hash = wavelet_hash(dataset["Original"])

for name, img in dataset.items():

    dist = hamming_distance(base_hash, h)

from sklearn.metrics import roc_curve, auc

# Gi lp nhn v im s (Khong cch nghch o)
# 1: Similar (Rotated, Noisy, Blurred), 0: Diff (Different)
# Ta thm chnh n (Original) vo nhm Similar
y_true = np.array([1, 1, 1, 1, 0]) # Original, Rotated, Noisy, Blurred, Different

# Score cng cao cng ging -> Score = max_dist - dist
# Gi s max_dist c th l len(hash)
max_possible_dist = len(base_hash)
y_scores = np.array([
    max_possible_dist - results["Original"],
    max_possible_dist - results["Rotated"],
    max_possible_dist - results["Noisy"],
    max_possible_dist - results["Blurred"],
    max_possible_dist - results["Different"]
])

fpr, tpr, thresholds = roc_curve(y_true, y_scores)
roc_auc = auc(fpr, tpr)

plt.figure()
plt.plot(fpr, tpr, color='darkorange', lw=2, label=f'ROC curve (area = {roc_auc:.2f})')
plt.plot([0, 1], [0, 1], color='navy', lw=2, linestyle='--')
plt.xlabel('False Positive Rate')
plt.ylabel('True Positive Rate')
plt.title('Receiver Operating Characteristic (ROC) - Demo')
plt.legend(loc="lower right")
plt.show()

from sklearn.metrics import accuracy_score, recall_score, precision_score

# Chn ngng (threshold) ti u t ROC (Youden's J statistic)
optimal_idx = np.argmax(tpr - fpr)
optimal_threshold = thresholds[optimal_idx]

# To nhn d on da trn ngng
y_pred = (y_scores >= optimal_threshold).astype(int)

accuracy = accuracy_score(y_true, y_pred)
recall = recall_score(y_true, y_pred)
precision = precision_score(y_true, y_pred)

print(f"Optimal Threshold: {optimal_threshold}")
print(" chnh xc (Accuracy):", accuracy)
print(" nhy (Recall):", recall)
print(" chnh xc (Precision):", precision)

IV> Kt lun

  • Hash Wavelet trch xut c trng tn s thp, rt bn vng vi cc bin i nhiu, m, v thay i sng.
  • Vi bin i xoay (Rotation), Hash Wavelet tiu chun km hiu qu hn (khong cch tng nhanh), cn cc k thut bt bin xoay (Rotation Invariant) nng cao hn hoc tin x l xoay.
  • Khong cch Hamming cung cp mt thc o n gin v hiu qu so snh.
